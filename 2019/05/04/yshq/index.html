影视后期制作专业术语科普 （个人笔记）

百科简介

视频（Video）泛指将一系列静态影像以电信号的方式加以捕捉、纪录、处理、储存、传送与重现的各种技术。

BDRip

关键词： BD
    释义: Blue-ray-Disc
            蓝色 光线 碟片
    波长对比(单位：纳米)       (读写用)激光精度≈波长的一半
    CD 780nm 激光                 即：蓝光200nm 红光325nm
    DVD 650nm 红光              同样大的盘，蓝光能读更小的点
    BD 405nm 蓝光                 点多了，记录的信息自然也就多了

关键词：Rip
    释义: Ripping
    猛烈的撕开 撕裂
    ①碟片→硬盘，这一过程
    ②原盘→压制
    ③词语滥用，大家都已经默认习惯了，
    对别人压制过的进行再压制也叫Rip

显示分辨率

释义:Display Resolution
1920 x 1080 = 2073600 (200W)
  (行)      (列)       (像素总数)

关键词：光敏感性癫痫症

(PSE： Photosensitive Epilepsy)
视觉神经受到强烈刺激影响到脑部的控制
轻则不舒服，重则会昏厥或者痉挛 全身抽搐

关键词：暗牧圣光（马赛克）

免费电视台：有和谐
收费电视台：无和谐

宽高比

({1920 \over 1080}) = ({16 \over 9})
